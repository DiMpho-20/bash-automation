# Bash Automation Toolkit

A collection of Bash scripts for automating common developer tasks.

## Scripts

### Beginner
- `hello.sh` — This is a simple program that greets a user then  displays they date ,time and file path they are in. 
- `project_generator.sh` — Program automates the folder structure of a project.
- `backup.sh` — project backups a folder

### File Management
- `organize_downloads.sh` — Program organizes files by their type
- `find_large_files.sh` — Program diplays the files that are over a certain size
- `clean_temp.sh` — Program deletes  .log and .tmp files

### Logs
- `error_counter.sh` — Program grabs all "ERROR" comments in a folder then sorts them.
- `log_analyzer.sh` — Program counts the num of errors,warnings and infos. 

### System
- `disk_usage.sh` — Program diplays Disk and Memory Usage
- `process_monitor.sh` — Program displays all current active processes then gives the user an option to kill a certain process

### Deployment
- `deploy.sh` — This program navigates to a folder and runs a git pull request to get latest code.

## How to Use
```bash
git clone https://github.com/DiMpho-20/bash-automation.git
cd bash-automation
chmod +x 
./
```
